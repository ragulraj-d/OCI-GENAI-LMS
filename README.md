# OCI-GENAI-LMS

Learning portal for the LTIMindtree OCI GenAI Foundation Programme — Batch: Kalinga, June 2026.

A 53-day corporate training curriculum covering AI, Machine Learning, Oracle Cloud Infrastructure (OCI), and Generative AI. Each day's session guide is a standalone HTML page, linked from the central index.

## Contents

- **Week 1–2:** SQL & PL/SQL fundamentals
- **Week 3:** Pandas, Machine Learning & Neural Networks
- **Week 4:** Deep Learning, NLP, LLMs & Deployment
- **Week 5:** Oracle APEX, OCI Gen AI, LangChain & LangGraph
- **Week 6:** AI Agents, RAG, OCI Storage & Compute
- **Week 7:** MCP Development, AI Ethics & Final Capstone

## Live Site

Hosted via GitHub Pages: https://ragulraj-d.github.io/OCI-GENAI-LMS/

## Structure

- `index.html` — course index with links to all available day guides
- `dayNN-*.html` — individual day/topic guides
